📚 E-Learning Platform

A scalable and feature-rich E-Learning Platform designed to deliver a complete online learning experience.
The system supports course management, assessments, live interactions, scheduling, and secure payments — built using Clean Architecture and modern backend technologies.
____________________________________________________________________________________________________________________________________________
🚀 Overview

The platform is designed around three main roles, each with its own dedicated dashboard:

Admin
Instructor
Student
____________________________________________________________________________________________________________________________________________
Each role has specific permissions and responsibilities to ensure a smooth and structured learning experience.

🧩 System Workflow
👤 Student Dashboard
____________________________________________________________________________________________________________________________________________
Students can:

Register and log in securely
Browse and enroll in available courses
Access all enrolled courses
Schedule and take quizzes and exams
Receive real-time notifications
Track their learning progress
Earn a certificate for each completed course
____________________________________________________________________________________________________________________________________________
👨‍🏫 Instructor Dashboard

Instructors can:

Create and manage courses
Organize course content (lessons/modules)
Add quizzes and exams
Manage assessments and student interactions
____________________________________________________________________________________________________________________________________________
🛠️ Admin Dashboard

Admins can:

Manage users and roles (Students, Instructors)
Monitor system activity
Control platform operations and overall performance

🏗️ Architecture

The project follows Clean Architecture:

Core Layer → Business logic & entities
Application Layer → Use cases & services
Infrastructure Layer → Database & external services
Presentation Layer → API controllers
