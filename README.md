📚 E-Learning Platform

A scalable and feature-rich E-Learning Platform designed to deliver a complete online learning experience.
The system supports course management, live sessions, assessments, and secure payments — all built using Clean Architecture and modern backend practices.

🚀 Overview

This platform enables instructors to create and manage courses, while allowing students to enroll, attend live sessions, take quizzes and exams, and receive certifications upon completion.

As a Team Leader, I led the development process, ensuring high code quality, scalability, and effective team collaboration.

✨ Key Features
🔐 Authentication & Authorization
Secure user registration and login system
Role-based access control (Admin, Instructor, Student)
JWT-based authentication
User profile and account management
📚 Course Management
Create, update, and delete courses
Organize course content into structured modules and lessons
Student enrollment system
Track student progress within each course
🎥 Live Sessions (Zoom Integration)
Integration with Zoom API for real-time learning
Schedule and manage live sessions within courses
Automatically generate meeting links
Allow students to easily join sessions
📝 Exams & Exam Attempts
Create exams with multiple question types
Configure time limits and grading rules
Track and store student exam attempts
Automatically calculate and store results
Apply validation rules to reduce cheating
❓ Quizzes & Quiz Attempts
Lightweight quizzes for quick assessments
Configurable multiple attempts
Instant feedback and automatic scoring
Track individual student performance
🎓 Certification System
Generate certificates upon course completion
Ensure all completion requirements are met (progress, exams, etc.)
Downloadable certificates
Certificate validation for authenticity
💳 Payment Integration (Stripe)
Secure online payments using Stripe
Support course purchases and enrollments
Handle payment success and failure workflows
Maintain transaction history
📊 Progress Tracking & Analytics
Track student progress within each course
Monitor performance in quizzes and exams
Admin dashboard for system usage and insights
🏗️ Architecture

The project follows Clean Architecture principles:

Core Layer: Entities and Business Logic
Application Layer: Services and Use Cases
Infrastructure Layer: External integrations (Stripe, Zoom, Database)
Presentation Layer: API Controllers
🔧 Key Design Patterns
Repository Pattern
Unit of Work
Dependency Injection
Separation of Concerns
🛠️ Technologies Used
Backend: ASP.NET Core Web API
Database: SQL Server
Authentication: JWT
Payments: Stripe API
Live Sessions: Zoom API
Architecture: Clean Architecture
